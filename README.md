# Cognifyz Task-2: Inline Styles, Client & Server Validation

## Overview
This project is part of my internship at Cognifyz Technologies.  
It demonstrates a registration form with **enhanced user interactions**, **inline styles**, **client-side validation using JavaScript**, **server-side validation using Node.js and Express**, and **temporary data storage in a JSON file**.

## Features
- **Inline Styles** for all form elements with hover and focus effects.
- **Client-Side Validation**:
  - Name must be at least 3 characters.
  - Valid email format.
  - Password must be at least 6 characters.
- **Server-Side Validation** for all submitted data.
- **Temporary Storage**:
  - Valid data is stored in `data/data.json`.
  - View stored data via `/submissions` route.
- **Smooth Transitions and UI Enhancements** using inline CSS and JavaScript.
- **Alert messages** for successful submissions.
